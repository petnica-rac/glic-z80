# glic-z80
Collection of assembly programms for the z80 in glic80 env.

In `src/` there is a number of tested z80 assembly programs which run on the `glic80` platform.
In `bin/` we place assembled binaries for each source code within the `src/` directory.

All binaries were created under Windows, with the `thiasm` assembler, version `0.45`. See http://www.tni.nl/products/tniasm.html
